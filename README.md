## 🚌 BusStation — Информационная система управления автобусным парком

Курсовой проект по дисциплине «Проектирование информационных систем».  
Разработка БД и клиентского приложения для автоматизации продажи билетов, управления рейсами и контроля посадки пассажиров междугороднего автовокзала «Северный» (Санкт-Петербург).

## 🎯 Цель проекта

Замена бумажного учёта и устаревшей кассовой системы на единую информационную систему, обеспечивающую:
- Централизованное управление 25 междугородними маршрутами
- Онлайн-бронирование и продажу билетов без двойных продаж
- Интерактивный выбор мест на схеме салона автобуса
- Контроль посадки и учёт неявившихся пассажиров
- Аналитическую отчётность по загрузке рейсов и выручке

## 🛠 Технический стек

| Компонент | Технология |
|-----------|-----------|
| СУБД | MS SQL Server 2022 |
| Клиентское приложение | WPF (.NET 7.0, C#) |
| Доступ к данным | ADO.NET (транзакции), Entity Framework (отчёты) |
| Моделирование | draw.io, PlantUML |
| IDE | Visual Studio 2022 |

## 📐 Архитектура базы данных

База данных содержит **8 нормализованных таблиц**:
- `Маршруты`, `Автобусы`, `Водители`, `Рейсы`
- `Места`, `Пассажиры`, `Кассиры`, `Билеты`

### Ключевые объекты БД:
- **VIEW `vw_СвободныеМеста`** — динамическая схема свободных мест на рейс
- **PROCEDURE `usp_ПродатьБилет`** — атомарная продажа с проверкой уникальности места в транзакции
- **TRIGGER `trg_TicketReturn`** — автоматическое освобождение места при возврате билета
- **Фильтрованный индекс `UX_Tickets_Trip_Seat`** — защита от двойных продаж на уровне СУБД
